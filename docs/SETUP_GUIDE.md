# מדריך הקמה — ListWise AI

מדריך זה מכיל **רק** את הפעולות שדורשות ממך פעולה אישית (זהות/כרטיס אשראי). כל שאר הקוד נבנה ומוגדר עבורך.

## שלב 1: חשבון Shopify Partner (חינם)

1. פתח את הקישור: https://partners.shopify.com/signup
2. הירשם עם כתובת האימייל שלך (או Google login)
3. מלא את פרטי הפרופיל הבסיסיים (שם, מדינה)
4. **אין צורך בכרטיס אשראי בשלב הזה**

בזמן שריצת ה-CLI `npm init @shopify/app` פועלת, היא תפתח דף התחברות עם קוד אימות בן 8 תווים (לדוגמה `QFCV-DJMH`) — יש להיכנס לקישור שמופיע בטרמינל ולאשר את הקוד עם חשבון ה-Partner שלך.

## שלב 2: חנות פיתוח (Development Store)

לאחר ההתחברות, ה-CLI ישאל אם ליצור אפליקציה חדשה בתוך הארגון שלך ואם לחבר חנות פיתוח. אם אין לך חנות פיתוח:

1. עבור ל-Partner Dashboard > Stores > Add store > Development store
2. בחר "Create a store to test and build" > תן שם (לדוגמה `listwise-dev`)
3. השאר את זה כחנות בדיקה בלבד (לא לחנות אמיתית)

## שלב 3: מפתח API ל-OpenAI

1. עבור ל-https://platform.openai.com/signup
2. הירשם עם אימייל
3. בתוך "Billing" הוסף אמצעי תשלום (כרטיס אשראי) - OpenAI גובה לפי שימוש (usage-based), לא מנוי חודשי קבוע. עבור נפח ההתחלה שלנו, מדובר בסביבות 0.001-0.01$ ליצירה אחת - כלומר כמה דולרים בודדים בחודש הראשון.
4. עבור ל-API Keys > Create new secret key
5. שמור את המפתח (מתחיל ב-`sk-...`) - נצטרך אותו בקובץ `.env`

## שלב 4: חשבון Hosting (Render.com מומלץ)

1. עבור ל-https://render.com ולחץ Sign Up (אפשר עם GitHub - אין צורך בכרטיס לרוב ה-Free/Hobby tier)
2. חבר את חשבון ה-GitHub שלך (או נעשה זאת בהמשך כשהקוד יהיה ב-repo)
3. אם תבחר בתוכנית בתשלום (מומלץ לפרודקשן, ~7$/חודש ל-Web Service + DB), תצטרך להזין כרטיס אשראי

## שלב 5 (מאוחר יותר, רק כשיש הכנסה): פרטי Payout

בתוך Partner Dashboard > Settings > Payouts, תזין את פרטי חשבון הבנק שלך לקבלת התשלומים מהמנויים. שלב זה רלוונטי רק לאחר שהאפליקציה חיה ויש התקנות.
